LeetCode 1： 两数之和

题目描述

给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。但是，你不能重复利用这个数组中同样的元素。

示例:
给定$nums = [2, 7, 11, 15], target = 9$

因为$nums[0] + nums[1] = 2 + 7 = 9$
所以返回$[0, 1]$

思想:

寻找$target-nums$，

题解

暴力破解

  • 用两层循环，时间复杂度$o(n^2)$
  • 使用List中相关函数，在每次循环中查找索引，本质上依然是$o(n^2)$的时间复杂度

solution1:

在每次循环中顺序查找,但当出现重复数字时，第一次查找只会找到自身，只有第二次查找时才能得到结果。

示例:
给定$nums = [3, 7, 11, 3], target = 6$

只有第二次查找时，才会返回结果

def twoSum(nums, target):
    for inx, num1 in enumerate(nums):
        num2 = target - num1
        if num2 in nums and nums.index(num2) != inx:
            return [inx, nums.index(num2)]
    return []

solution2:

$num2$的查找并不需要每次从$nums$查找一遍，只需要从$num1$位置之前或之后查找即可。

def twoSum(nums, target):
    for inx, num1 in enumerate(nums):
        num2 = target - num1
        temp = nums[:inx]
        if num2 in temp:
            return [nums.index(num2), inx]
    return []
